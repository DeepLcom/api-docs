---
title: "New: XML/HTML handling v2"
description: "Information on the improved version 2 of tag-handling translations"
---

Tag handling v2 is an improved algorithm for translating XML and HTML content. Set `tag_handling_version` to `v2` to enable it.

## Key differences

- **Output changes**: v2 produces different results than v1. Test your system before switching.
- **Strict XML parsing**: Invalid XML returns an error.
- **Model compatibility**: v2 requires the <a href="/api-reference/translate#param-model-type">quality-optimized model type</a>. Setting `tag_handling_version` to `v2` automatically sets `model_type` to `quality_optimized`.

## Default behavior

- **New customers** (after 1 December 2025): Default to v2
- **Existing customers**: Continue to default to v1

<Warning>
  Requests with `model_type=latency_optimized` and `tag_handling_version=v2` return an error.
</Warning>
