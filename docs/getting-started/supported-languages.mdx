<Info>
  Each language shows which features it supports as tags: **Translation**, **Glossaries**, **[Tag Handling](/docs/xml-and-html-handling/xml)**, and **[Text Improvement](/api-reference/improve-text)**. Languages without a tag for a given feature do not support it. Languages tagged **Translate: Target Only** (typically regional variants like `EN-GB` or `PT-BR`) can only be used as target languages. Use the feature filters to find languages with specific capabilities.

  To convert between variants of the same language (e.g. `EN-US` to `EN-GB`), refer to [this guide](/docs/learning-how-tos/examples-and-guides/translating-between-variants). Only fully-featured languages are returned by the [`/languages` endpoint](/api-reference/languages/retrieve-supported-languages). [Text improvement](/api-reference/improve-text) languages have not yet been added to that endpoint. For `/write/rephrase`, `writing_style` and `tone` currently work in `DE`, `EN-GB`, and `EN-US`.
</Info>
