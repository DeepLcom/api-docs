## Language-Specific Information

Chinese has simplified and traditional characters, plus regional variations. Style rules help you choose the appropriate character set and style.

## Common Use Cases

- Simplified vs Traditional characters
- Business communications (formal tone)
- Academic writing (formal register)
- Technical documentation (clear language)

## Getting Started

1. **Identify your needs** - What type of content are you translating?
2. **Choose relevant rules** - Focus on rules that affect your content type
3. **Test different options** - Try various combinations to find what works best
4. **Apply consistently** - Use the same rules across all your content
