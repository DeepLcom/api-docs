## Language-Specific Information

Japanese has multiple writing systems (hiragana, katakana, kanji) and formality levels. Style rules help you choose appropriate characters and tone.

## Common Use Cases

- Business communications (appropriate honorifics)
- Academic writing (formal register)
- Technical documentation (clear, precise language)
- Marketing content (appropriate formality)
