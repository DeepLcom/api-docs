## Language-Specific Information

French has formal and informal registers, plus regional variations. Style rules help you maintain appropriate formality levels.

## Common Use Cases

- Business communications (formal vous address)
- Academic writing (formal register)
- Technical documentation (clear, precise language)
- Marketing content (appropriate formality)
