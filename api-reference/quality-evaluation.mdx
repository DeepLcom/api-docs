## Overview

Identify quality issues with your translations through a two-step process:

1. [**Submit an evaluation job**](/api-reference/quality-evaluation/submit)
2. [**Poll for the result**](/api-reference/quality-evaluation/poll)

## Limits

| **Limit** | **Value** |
|---|---:|
| Segments per request | 500 |
| Characters per segment (`source` or `target`) | 10,000 |

### Rate limits

| **Endpoint** | **Limit** |
|---|---|
| `POST /v1/quality-evaluation` | 100 requests per minute per API key |
| `GET /v1/quality-evaluation/{job_id}` | 1,000 requests per minute per API key |

Exceeding either limit returns `429 Too Many Requests`.

## Job retention

Job results are retained for 24 hours after the job reaches `done` or `error` state. After that, the `GET` endpoint returns `404 Not Found` for that `job_id`.
